modes i: insert
s: seek
: command 
v: visual
z: z mode for scrolling
f: find mode
space: space mode

: command mode
- tab and shift-tab to pick from selection
- space to pick or ctrl+y 
- number: go to line number
- e: reload window

visual mode
o: move cursor to ends of selection
V: line wise visual mode
ctrl+v: block wise visual mode
I/A + block selection: add to each line of block

normal mode
o: insert new line
O: insert new line above
shift+$: move cursor to end of line
number + hjkl: move cursor number lines/characters
w: next word
e: end of word
b: backwards word
WEB: move to next/end/backwards whitespace delimited word ignoring special characters
A: end of line
I: beginning of line
^: move cursor to beginning of first word
ctrl+o/i: jump to previous/next cursor position
number-i-text-escape: insert text number times
c: like delete but enter insert mode
D/C: delete to end of line, C enters insert mode
x: delete character under cursor
J: join lines
gUmotion: uppercase line/word
gUaw: uppercase inside word with a for around
~: toggle case of character
.: repeat last command
qq: start recording command, q to stop recording
(): navigate by sentence / paragraph
{}: navigate by paragraph explicitly
%: jump to matching bracket backwards or forwards or next open bracket
[{: backwards match curly bracket from inside block
[%/]%: jump to matching bracket inside block
[[: jump to prev reference of a variable or function
]]: jump to next reference of a variable or function
]i/[i: go to matching indentation
]d/[d: navigate to diagnostics, replace with e for errors and w for warnings
[h/]h: jump between git changes
c number i/a object: will delete surrounding number objects
n / l in operation mode to select next previous object without need for navigation
operator pending mode + S enter seek mode for objects
gsaa/i object character: surround append to add brackets, quotes, etc.
" letter + motion: save letter register whatever was deleted / yanked / changed
"0: last yank register
"%: filepath
]p/[p: paste above or below with indentation
p[y...: paste and go through registers
H/L: selecte left/right buffer, works with count
space bb: switch to other buffer
space comma: buffer picker
esc in input field in picker switches to selection window 
space .: scratch buffer
space bo: close other buffers
space bl: close buffers to the left
space br: close buffers to the right
space wh/l/j/k: move to window left, right, down, up
ctrl+h/l/j/k: move to window left right down up
space wd: delete window
space wv: split window with current content
space ws: split window horizontally
ctrl+v: when selecting files, open in vertical split
ctrl+s: when selecting files, open in horizontal split
space+w+space: open multiple windows in a row,
space tab tab: open new tab
gt/gT: next previous Tab, works with numbers 
3gt: select tab 3 
zc: collapse current fold
zo: open current fold
za: toggle fold
zR: open all closed folds
space qq: close all
space qS: show all sessions
space snh: notification history
space uw: enable line wrap
space ft: terminal
space+ww: cycle windows
paste in terminal: ctrl+shift+v

z mode
z enter or zt: scroll current line to top
zz: scroll line to center
zb: scroll line to bottom

f/t mode
f + characters: highlight searched chars
number 
t + characters: highlight same as f
number + t: move cursor to before n instance

file picker (space space)
alt+s shows letters to pick file before paths, allows to seek
tab selects multiple files
space + f F: pick files from cwd
ctrl+f/b scroll preview 
ctrl+d/u scroll file list d down an u up

explorer
space + e open root directory
space + E open cwd
d when file is selected to delete file
a to create new file or folder when ending in /
r to rename
y/p to yank and put a file somewhere else
m move file to cursor location after selecting with tab
P toggle preview
ctrl+T terminal

plugins
flash.nvim: cursor navigation by searching letters 
snacks.nvim: lots of small plugins
:LazyExtras to list plugins, x to install or uninstall depending on status
